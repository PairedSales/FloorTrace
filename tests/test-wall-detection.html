🎯 Topology-Guided Detection Strategy

Goal: Build a complete wall adjacency graph with high recall (detect ALL walls)
1. Detect: Use low edge threshold (3-5%) to get many candidate line segments
2. Merge: Build topology by merging collinear segments (handles dashed walls)
3. Fill: Connect gaps at doors/windows to complete wall chains
4. Prune: Remove noise, duplicates, and short artifacts
5. Classify: Label exterior vs interior walls based on connectivity

⚡ Key Tuning Tip: If missing interior walls, LOWER Edge Threshold % (try 3%). If too much noise, raise Min Wall Length or enable stricter post-processing.
